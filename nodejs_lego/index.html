Introduction

Nodejs 乐高

一个全栈摩登 JS 开发者的必备 nodejs 基础，服务正在使用 React/Meteor/Express 这些框架进行开发的朋友们。初学 Nodejs 开发的小白，以及开发 Nodejs 一段时间，但是其实希望更深入的了解一下 Nodejs 的一些底层机制的老手。帮助大家理解何谓非阻塞，异步 IO ，模块化编程，HTTP 服务器这些 Nodejs 的基础知识。另外，每天安装 NPM 包是我们最重要的事情了，所以课程中会自己手动制作发布一个 npm 包，加深我们对 npm 工作方式的理解。

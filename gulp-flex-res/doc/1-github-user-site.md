# 搭建 Github 用户网站

参考 <http://pages.github.com> 。可以创建“用户网站”，也可以创建“项目网站”，本视频介绍用户网站的搭建方式。


### 托管方式

直接把 html/css/js 存放到 happypeter.github.io 这个仓库的 master 分支之上。
这样就可以在 <http://happypeter.github.io> 访问到托管的内容了。

# 进一步学习建议

后续的学习关键词，以及学习顺序如下：

- module 学习使用 ES6 模块
- webpack 使用 webpack 进行模块打捆，没有 webpack 模块化的项目根本不能直接运行
- react 前面 ES6 的知识都有了，就可以实际的去学习 react 框架了，上手实际项目开发

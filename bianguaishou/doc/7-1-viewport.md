# Viewport 禁用缩放

移动设备会把页面自动缩小，为了让我们设置的 CSS 长度跟设备的像素长度对应，我们首先要做的就是禁用缩放，这个可以通过添加 viewport 设置来实现。

代码如下：

```
<meta name="viewport" content="width=device-width, initial-scale=1" />
```

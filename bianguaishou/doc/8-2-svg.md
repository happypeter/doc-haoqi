# 使用 SVG 图标

传统上使用 png 作为图标的形式因为放大之后会失真，所以不太适合做响应式页面。于是，矢量化图标变得流行，一种方式是使用类似 font-awesome 这样的“字体图标”，
另外一种就是使用 SVG 图标。本节我们介绍 SVG 图标的使用。

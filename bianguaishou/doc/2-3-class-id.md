# class 和 id 的妙用

class 和 id 都是用来标记一个元素的，方便我们在 CSS/JS 中准确的选择元素。差别在于，同一个 class 名，当前页面中可以出现多次，但是同样一个 id 名，在同一个页面中只能出现一次。

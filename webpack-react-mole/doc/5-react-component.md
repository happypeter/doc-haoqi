### 使用 React 的前提条件

要有 Babel ，到 [Babel 官网](https://babeljs.io/) ，可以看到 Babel 的主要功能，除了可以编译 ES6 ，就是可以编译 JSX 代码。

另外要保证安装两个包 react 和 react-dom 。

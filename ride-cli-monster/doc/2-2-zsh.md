# Oh My Zsh

前面我提过了，Bash 是开发者的命令行，Mac/Linux 上默认都是 Bash 。实际情况是，用 Mac 的开发者，通常都会对 Bash 命令行做一些优化，不改变 Bash 的基本使用方式，只是让它变得更好看也更好用，这种优化方式就是安装 [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) 。

### 可以达成的效果

- 更好看的命令行提升符。
- git 分支展示
- 信息高亮
